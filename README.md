# 📚 BOOXCHANGE

## About
BOOXCHANGE is a simple book exchange listing website built in 2021 as a final study project. It was developed using native PHP and basic MySQL database integration.


The project allows users to register, log in, and publish or view book listings for exchange purposes.

---

## 🛠️ Technologies Used

- PHP (Native)
- HTML / CSS
- MySQL
- JavaScript

---

## 📁 Project Structure

- Database/ → Database files  
- Profile/ → User profile pages  
- Verification/ → Basic account verification  
- assest/ → Static files (CSS/images)  
- Pofile-user.php → User profile view  
- home.php → Home page  
- index.php → Landing page  
- login.php → Login system  
- reg.php → User registration  
- profile.php → Profile management  
- resetpassword.php → Password reset system  
- clear.php → Utility script  

---

## 📌 Features

- User registration and login  
- Basic book listing system  
- User profile pages  
- Simple password reset  
- Basic verification flow  

---

## ⚠️ Note

This is a very basic and early-stage project built for learning purposes.


## 💡 Project Purpose

The goal of BOOXCHANGE is to create a simple system where users can:
- Register and create an account
- Share books they want to exchange
- View other users’ available books
- Connect users through basic book exchange functionality
